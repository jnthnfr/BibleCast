    Speech engine
    Vosk gives real-time word-by-word output (downloads ~45 MB model once, fully offline). Whisper AI is more accurate but transcribes in 2–3 second chunks.
    Whisper model size
    Larger models are more accurate but slower to load and use more RAM.
    AI Performance
    CPU threads
    More threads = faster transcription on multi-core systems. "Auto" uses half your CPU cores (recommended).
    GPU acceleration (WebGPU)
    Runs Whisper AI on your GPU — significantly faster on supported hardware. Falls back to CPU if unavailable.

    Voice Commands
    AVAILABLE COMMANDS
    "next / next verse"Advance to verse +1 in the same chapter
    "previous / previous verse / go back"Go back to verse −1 in the same chapter
    "clear / clear screen / clear the screen"Hide the display output
    "hide / hide screen / hide the screen"Hide the display output
    "show / show verse / project / project verse"Show the current verse on display
    "repeat / repeat that / repeat verse"Re-project the current verse

    Voice commands are detected in live transcription and only activate when Voice Cmds is enabled in the top bar and a session is active.
